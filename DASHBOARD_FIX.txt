🧪 HOW IT WORKS NOW
═════════════════════════════════════════════════════════════════════════════════

User Flow:
  1. User enters URL on Dashboard
  2. Clicks "Analyze URL" button
  3. Frontend sends to Flask backend (localhost:5000/api/predict)
  4. Backend extracts URL features
  5. Backend runs ML model
  6. Backend returns prediction + confidence
  7. Frontend displays results on Results page

API Request:
  POST http://localhost:5000/api/predict
  {
    "url": "https://example.com"
  }

API Response:
  {
    "url": "https://example.com",
    "is_phishing": false,
    "confidence": 98
  }

Results Display:
  ✅ URL: https://example.com
  ✅ Status: Safe (or Phishing)
  ✅ Confidence: 98%
